# Automatización con Agentes en Google Gemini

**Curso completo de 0 a Experto**

## Descripción

Este curso te llevará desde los fundamentos de IA generativa hasta la construcción de sistemas multi-agente complejos usando Google Gemini. Aprenderás a crear agentes inteligentes que pueden razonar, usar herramientas, mantener memoria, y colaborar entre sí.

## Requisitos

### Software

- Python 3.10+
- pip o poetry
- Git

### Cuentas

- Google Cloud Platform (GCP)
- API Key de Gemini

### Conocimientos previos

- Programación en Python (intermedio)
- Conceptos básicos de APIs REST
- Familiaridad con JSON

## Inicio Rápido

```bash
# 1. Clonar el curso
cd teach-laoz-curso-gemini-agents

# 2. Crear entorno virtual
python -m venv venv
source venv/bin/activate  # Linux/Mac
# o: venv\Scripts\activate  # Windows

# 3. Instalar dependencias
pip install google-generativeai python-dotenv

# 4. Configurar API key
cp .env.example .env
# Editar .env con tu GOOGLE_API_KEY

# 5. Verificar instalación
python -c "import google.generativeai; print('OK')"
```

## Perfil de Egreso

Al completar este curso serás capaz de:

1. Diseñar e implementar agentes inteligentes con Google Gemini
2. Construir sistemas multi-agente con orquestación y colaboración
3. Integrar agentes con fuentes externas mediante RAG y Function Calling
4. Desplegar agentes en producción con seguridad y observabilidad
5. Evaluar y optimizar el rendimiento de sistemas de agentes

## Proyecto Final

Sistema multi-agente para automatización empresarial que incluye:

- Agente coordinador (supervisor)
- Agentes especializados (investigación, análisis, ejecución)
- Sistema de memoria persistente
- Integración RAG
- API REST y dashboard de monitoreo
